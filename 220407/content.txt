ㅁ 함수
 - 함수 반환형, 매개변수 목록의 형식을 잘 지켜야함. 다른것을 넣으면 오류남.

ㅁ 함수 선언
 - 함수를 호출할것을 선언 ( 메인 함수위에 선언 )
 - 그 함수를 메인함수 밑에 정의함.

ㅁ 형식매개변수(formal parameters)
 - 함수가 정의될때의 매개변수
 -  선언 : int max(int a, int b)

ㅁ 실매개변수 or 실인자
 - int max = max(a, b)

ㅁ 값에 의한 호출
 - 함수가 호출 될시 실인자의 값이 형식인자의 변수에 각각 복사된 후 함수 실행

ㅁ 함수의 매개변수로 배열을 전달
 - 대부분 매개변수로 배열을 줄때 인자값을 배열의 크기로 하나 더 준다
	- 그 이유는 함수안에서는 배열의 크기를 알수없기 때문이다.
	- 함수 선언이나 몸체에다가 sizeof~~~해도 알수없다는 말임.

 - double ary[] == double *ary와 같다 / 시작주소를 말함.
 - 함수 호출 시 배열이름으로 배열인자를 명시한다.
 - data] ={1,2,3,4} 일대, data 는 data[0]의 주소와 같다. ****
